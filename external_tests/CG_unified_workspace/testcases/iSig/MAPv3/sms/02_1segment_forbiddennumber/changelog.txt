Short Description:
===================
This iSig TC simulates a single segment SMS submission
for a forbidden destination number 


Flow details:
==============
Since this TC is made up by a single request/response pair
there's no specific description necessary

Parameters: 
============
  mscAddress             of the originatig Party (callSubscriber()) 
  smscAddress            of the originatig Party's SMSC 
  smsDestinationMsisdn   must be a forbidden number configured on the SMS service
  smsUserDataMessagePart
  smsUserDataLength
  smsHeaderPresentInUserData
  smsDataCodingScheme
  smsUserDataDecoded
